# Andrew's garden
This is my [digital garden](https://www.technologyreview.com/2020/09/03/1007716/digital-gardens-let-you-cultivate-your-own-little-bit-of-the-internet/), a place for thoughts. It's an [Obsidian](https://obsidian.md) vault published with [Quartz v4](https://quartz.jzhao.xyz) at [a.ndr.ooo](https://a.ndr.ooo).

It can be run locally with `npx quartz build --serve`.
