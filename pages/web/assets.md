# Web assets and tools

Some assets, mostly free and open-source, which you may want to add to your website. It's mostly **fonts**, because I love **fonts**. Also a few tools and generators.

## Fonts

### Header
- [Fraunces](https://fraunces.undercase.xyz/) is the title font on this site
- [Besley](https://indestructibletype.com/Besley.html) is excellent and so is the [Small Caps variant](https://github.com/ctrlcctrlv/Besley-Small-Caps)
- [ZT Bros Oskon 90s](https://zelowtype.gumroad.com/l/ztbrososkon90s) (partially paid)

### Serif
- [Crimson Pro](https://fonts.google.com/specimen/Crimson+Pro)

### Sans-serif
- [Atkinson Hyperlegible](https://www.brailleinstitute.org/freefont/) is what you're reading right now
- [Alan Sans](https://typeface.alan.com/)
- [Inclusive Sans](https://www.oliviaking.com/inclusivesans/feature)
- [Bricolage Grotesque](https://ateliertriay.github.io/bricolage/)
- [Rumiko Sans](https://www.typemates.com/fonts/rumiko-sans) (paid)

### Monospace (for code)
- [Atkinson Hyperlegible Mono](https://fonts.google.com/specimen/Atkinson+Hyperlegible+Mono) is the code font on this site
- [Hermit](https://pcaro.es/hermit/)
- [Operator](https://www.typography.com/blog/introducing-operator)
- [Victor Mono](https://rubjo.github.io/victor-mono/)

### Unique
