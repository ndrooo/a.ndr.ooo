# Making websites

A loose and occasionally updated collection of **resources** I've found helpful in building hobby and personal websites.

For more on how *this website* is made, check out [the colophon](/about#website).

<section-card title="Inspiration zone" href="./inspiration">

A curated and organized collection of some websites that I think are interesting, stylish, or unique. Sorted by style so you can borrow or steal as needed *(and as dictated by license terms)*.

</section-card>

<section-card title="Tools and assets" href="./assets">

The fun stuff: fonts, generators, components, snippets, themes, templates, etc. Mostly free, mostly open source, a few things in there that are paid but only when they're really good.

</section-card>

<section-card title="Further reading" href="./learning">

Tutorials and other informational resources I like, covering: how to build websites, suggestions for content, and more.

</section-card>
