## How to build your website
- [Every Layout](https://every-layout.dev/) - partially paywalled, but one of the best CSS layout guides I've encountered
- [Piccalilli](https://piccalil.li/) - very good HTML/CSS/JS content if you're coding your own site (some is paywalled but the free stuff is great)
- [Flexbox Froggy](https://flexboxfroggy.com/) - this is how I learned flexbox, and I go back to it frequently if I am trying to figure out a tricky flexbox thing
